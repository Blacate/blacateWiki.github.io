# Neail's wiki

```bash
yarn global add https://github.com/Blacate/MinoriWiki.git
git clone git@github.com:Blacate/wiki.git
cd wiki
git clone https://github.com/Blacate/MinoriWiki-theme-semantic-ui.git themes/semantic-ui # install theme
minori note # add new note
make publish # publish wiki & backup wiki
```
